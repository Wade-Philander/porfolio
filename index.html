My Projects

Python Projects

Tk. Weather App

Fetching my data from a Weather API, I have converted the json data using the knowledge I have learnt in Python. I am displaying the requested information from the user, using a built in function known as Tkinter.

Tk. Lotto Draw

My lotto game is an a application designed with Tkinter in Python which allows the user to enter their chosen lucky numbers ranging between 0-50. The application will fetch 6 random numbers once again between 0-50 and dependin how many correct numbers the user has chosen, they would recieve a prize accordingly. All attempts are also recording on a .txt file after the game for analysis.

Tk. Media Player

This is a Media player application created in Python using a built in method Tkinter. This allows any sound wave in the format of .wav files to be played on the users device. This application allows the user to add .wav files from their device into the media player as well as manipulate the sound in terms of stopping and playing the media device.
